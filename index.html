Therapy Linker API

This online counseling management system API enables users to book sessions and consult therapists for mental health services.

Emmanuel Kibet

CRUD Functionalities:

Create: Book an appointment with a therapist.
Read: Retrieve existing records from the database, view available bookings, therapists, and patients.
Update: Modify existing records, update user profiles.
Delete: Remove outdated reservations or discontinued records.
